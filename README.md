## 🌟 Key Pillars

### 🔐 Multi-Layer Verification
* **CNN-Powered Analysis:** Mock Convolutional Neural Network analyzes certificate authenticity.
* **Blockchain Hashing:** Secure SHA-256 anchoring via `ethers.js` for tamper-proof records.
* **Public Verifier:** Instant validation of any certificate using its unique hash.

### 🤖 AI Intelligence
* **Gemini Career Coach:** Real-time profile optimization and navigation assistance.
* **Verified Job Matching:** Matches your **verified** skills to live job opportunities.

### 📄 Professional Suite
* **Smart Resume Export:** One-click PDF generation with a "TrustSeal" digital signature.
* **Dual-Role Access:** Dedicated experience for both Talent Seekers and Employers.

## 🚦 Quick Start

### 1. Clone & Install
```bash
git clone https://github.com/Vin07h/TrustChain.git
cd TrustChain
npm install
```

### 2. Environment Setup
Create a `.env.local` and add:
```javascript
VITE_GEMINI_API_KEY = "your_key_here"
VITE_FIREBASE_CONFIG = "your_config_here"
```

### 3. Run Development
```bash
npm run dev
```
